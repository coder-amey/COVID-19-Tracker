How prediction works

The data-points are fitted to an exponential curve (y = abx). The discovered values of parameters 'a' and 'b' are used to predict the next expected value. As the situation is rapidly evolving, the characteristics of the curve change rapidly. Thus, only the last three data-points are used to fit the curve using exponential regression for an optimal prediction.
